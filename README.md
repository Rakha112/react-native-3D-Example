## Run Locally

Clone the project

```bash
git clone https://github.com/Rakha112/react-native-3D-Example
```

Go to the project directory

```bash
cd react-native-3D-Example
```

Install dependencies

```bash
yarn
```

Run the app

```bash
yarn expo start -c
```
